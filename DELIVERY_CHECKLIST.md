## 📝 交付说明

### 使用方式

1. **快速启动**
   ```bash
   ./run.sh
   ```
   或
   ```bash
   python3 app.py
   ```

2. **访问应用**
   打开浏览器访问: http://localhost:5000

3. **查看用户画像**
   在 Dashboard 页面向下滚动到"👤 用户画像分析"部分

### 核心特性

1. **智能标签系统** - 自动识别用户行为特征
2. **多维度分析** - 应用、时段、安全三维度
3. **交互式可视化** - 点击查看详细数据
4. **实时数据分析** - 上传新数据自动重新分析
5. **完整 API** - JSON 格式数据接口
